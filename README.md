# uprely

Uprely es un proyecto de monitoreo de uptime con frontend en React y backend en NestJS.

## Estructura

- `frontend/`: aplicacion cliente con Vite + React
- `backend/`: API principal con NestJS
- `nginx/`: configuracion de proxy

## Inicio rapido

Cada carpeta principal tiene su propio `package.json`, por lo que las dependencias se instalan por separado en `frontend/` y `backend/`.
